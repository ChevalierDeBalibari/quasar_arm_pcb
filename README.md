# quasar_arm_pcb
Altium Designer project of metal detector "Quasar ARM" 

Проект печатной платы металлоискателя "Quasar ARM" в Altium Designer.

Цели проекта:

1. Ознакомится с системой автомтизированного проектирования Altium Designer;
2. Разработать удобную для изготовления печатную плату металлоискателя "Quasar ARM":
   - использование smd-компонентов типоразмера 1206;
   - изготовление печатной платы по ЛУТ.
