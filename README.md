# quasar_arm_pcb
Altium Designer project of metal detector "Quasar ARM" 

Проект печатной платы металлоискателя "Quasar ARM" в Altium Designer.

Основной проект металлоискателя "Quasar ARM" находится по адресу:
http://fandy.ucoz.org/publ/metalloiskatel_quot_kvazar_quot_quot_quasar_quot/metalloiskatel_quot_quasar_arm_quot/2-1-0-5

Цели проекта:

1. Ознакомится с системой автоматизированного проектирования Altium Designer;
2. Разработать удобную для изготовления печатную плату металлоискателя "Quasar ARM":
   - использование smd-компонентов типоразмера 1206;
   - изготовление печатной платы по ЛУТ.
